Git 基本概念介绍：
    工作区 -> 缓存区 -> Local Repository -> Remote Repository 
    
    工作区：就是当前正在编辑的版本
    缓存区：从工作区提交，但是还没有放入到本地仓库的版本
    本地仓库：存储在本地仓库中的版本
    远程仓库：存储在远程仓库中的版本，比如：存储在 GitHub 服务器中的版本 
    

Git 中文件状态的改变，例如一下操作仅针对一个文件 H.txt
      未跟踪           未修改             已修改            缓存区              本地仓库            远程仓库
    untracked       unmodified          modified          staged            localRepo           remoteRepo
        |                |                  |               |                   |                   |
        |  - git add ->  |                  |               |                   |                   |
        |                |  - edit file ->  |               |                   |                   |
        |                |                  |  - git add -> |                   |                   |
        |                |                  |               | - git commit ->   |                   |
        |                |                  |               |                   |  - git push ->    |
        |                |                  |               |                   |  <- git pull -    |
        |                |             <- git checkout <- git reset -           |                   |
        |                |   <- git checkout -- H.txt -     |                   |                   |
        |  <- git rm -   |                  |               |                   |                   |
