# 解决 chrome 运行时弹出创建默认密钥环的问题

# Chrome Debug 中 Source 中源码格式不对

    Console 中 Ctrl+Shift+F 查找

# chrome 关闭同源策略
    在 chrome 启动参数中添加 --args --disable-web-security --user-data-dir  即可
    例如：shell> chrome --args --disable-web-security --user-data-dir
